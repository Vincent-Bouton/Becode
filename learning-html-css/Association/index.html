Z event

Z Event est un projet caritatif créé par Adrien Nougaret et Alexandre Dachary. Le but est de rassembler plusieurs animateurs spécialisés dans le streaming de jeu vidéo pour un marathon de plus de 50h, afin d'encourager les spectateurs à se mobiliser pour soutenir une association caritative.
